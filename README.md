# CV Web - Fernando Folcher

Este proyecto es un CV técnico desarrollado en HTML y CSS.

## Descripción

Se trata de una página web simple, moderna y responsive donde presento mi perfil profesional, tecnologías y proyectos académicos.
El diseño utiliza HTML semántico y estilos en CSS separados, con una estética limpia.

## Tecnologías utilizadas

- HTML5 (estructura semántica)
- CSS3 (diseño, grid, responsive y variables)

## Funcionalidades

- Presentación de perfil profesional
- Lista de tecnologías utilizadas
- Sección de proyectos con enlaces a GitHub
- Información de contacto
- Diseño responsive adaptable a dispositivos móviles
- Estilo moderno tipo portfolio
